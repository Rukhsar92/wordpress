A theme for restaurant and Cafe called Dinner Lite.

== Description ==

Dinner Lite is beautiful, clean and professional WordPress theme for restaurants, cafe, coffee house or any food drinks related business. Its easy to use, polished, feature-rich and fully responsive which looks perfect on any devices like mobile, pc and tablet and cross browser compatible. With its customizer options, one can easily set up homepage section perfectly without any hassle. You can check theme demo at: https://astaporthemes.com/demo/dinner-lite

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

Dinner Lite includes support for Infinite Scroll in Jetpack.
